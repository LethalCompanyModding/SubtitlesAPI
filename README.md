# SubtitlesAPI Community Package

A centralized API for fetching localized subtitles. This is a unified project merging and maintaining the packages formally known as Subtitles, AudibleDistanceLib, and SubtitlesAPI. See their respective projects for more information.

- [AudibleDistanceLib](AudibleDistanceLib)
- [Subtitles](Subtitles)
- [SubtitlesAPI](SubtitlesAPI)

## Feedback

Got any feedback or found bugs? Please do post it in [the Github Issues page](https://github.com/LethalCompanyModding/SubtitlesAPI/issues)!
